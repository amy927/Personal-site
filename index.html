Mock InstaCart via Windows Forms

Working with a partner, we made a mock InstaCart application via C# Windows Forms .NET Framework. This application included a log in screen, leading to either an inventory manager, customer, or a delivery driver depending on which account is signed in. A customer can make an order, once the order is placed (goes through checkout), the order will then pop up on the delivery-side of the application. The driver can see which items are needed for the order, see the address and customer name, and can mark the order as complete. The inventory manager aspect of the application can view all items the store has in stock and can adjust the quantities, add new items, and delete items.

I can't show this project via GitHub or online, but can run this for individuals if interested!
